- 내용의 내용들을 달고 싶으면 `-`로 특정 내용을 작성한 뒤, indent를 하고 다른 기호 `+`, `*` 등으로 쓸 것.
- 표 format
```
|제목|내용|설명|
|------|---|---|
|테스트1|테스트2|테스트3|
|테스트1|테스트2|테스트3|
|테스트1|테스트2|테스트3|
```
- 마크 다운에서 내부 링크로 이동할 때, `.`은 생략해도 되고, 띄어 쓰기는 `-`로 대체한다.
